MUSE

Accessible

Muse gives those with visual difficulties the opportunity to learn core programming concepts in an intuitive way.

Innovative

Instead of manipulating visual components, you can write scripts in our custom language, Calliope, in order to manipulate sounds to represent common data structures and algorithms.

Calliope

Calliope is written from the ground up to be very human readable so you can focus less on language syntax and more on programming.
